Theory and Formula Guide

Convolution Definition

Convolution computes the output of a linear time-invariant system from an input and an impulse response.
y(t) = integral x(tau) h(t - tau) dtau
It accumulates weighted overlap between one signal and a shifted version of the other.

Physical Interpretation

At each time instant, you measure how strongly the two signals align and sum this effect over all tau.
Discrete idea: y[k] = sum x[n] h[k - n]
In this lab, convolution is computed numerically using sampled values over a finite range.

Connection to Symmetry

Even and odd decomposition helps analyze why certain convolution outputs appear symmetric or skewed.
x_e(t) = (x(t) + x(-t)) / 2
x_o(t) = (x(t) - x(-t)) / 2
Decomposing first can make convolution behavior easier to reason about component-by-component.
