Theory and Formula Guide

Amplitude Scaling

Scaling changes the vertical magnitude of the signal without changing its time axis shape.
y(t) = A x(t)
If |A| > 1, the waveform grows. If 0 < |A| < 1, it shrinks. A negative A flips polarity.

Time Shift and Reversal

Time operations move or mirror the waveform along the horizontal axis.
Shift: y(t) = x(t - t0)
Reverse: y(t) = x(-t)
Positive t0 delays the signal, negative t0 advances it, and reversal mirrors it around t = 0.

Link to Decomposition

Operation results can be split into symmetric and antisymmetric parts for deeper analysis.
x_e(t) = (x(t) + x(-t)) / 2
x_o(t) = (x(t) - x(-t)) / 2
This helps interpret how each operation affects symmetry in the transformed signal.
