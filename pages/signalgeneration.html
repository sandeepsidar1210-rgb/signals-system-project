Theory and Formula Guide

Signal Definition

A signal is a function of time that carries information, such as voltage, sound, or motion data from a real system.
x(t): continuous-time signal
In this lab, you generate and inspect standard signals to understand how shape and parameters affect behavior.

Elementary Signals

Three common building blocks are sine, step, and ramp signals.
Sine: x(t) = A sin(2 pi f t)
Step: u(t) = 1 for t >= 0, else 0
Ramp: r(t) = t for t >= 0, else 0
These are frequently used to model periodic behavior, switching events, and linear growth.

Core Extensions

Generated signals can be transformed or combined in later analysis modules.
Even: x_e(t) = (x(t) + x(-t)) / 2
Convolution: y(t) = integral x(tau) h(t - tau) dtau
These formulas connect generation with decomposition and system-response analysis.
